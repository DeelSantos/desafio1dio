O que é C#?

C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft. Ela foi criada para desenvolvimento de aplicativos para a plataforma Windows e é usada para desenvolver uma ampla variedade de aplicativos, incluindo aplicativos de desktop, web, mobile, games e muito mais. C# é uma linguagem de alto nível e fácil de aprender, com uma sintaxe similar a outras linguagens populares, como Java e C++.

Por que aprender C#?

C# é uma das linguagens de programação mais populares e poderosas do mundo. Ele é amplamente utilizado na indústria de desenvolvimento de software e tem uma grande comunidade de desenvolvedores. Além disso, o C# é uma das principais linguagens de programação usadas para desenvolver aplicativos para a plataforma Windows, o que significa que existem muitas oportunidades de trabalho para Devs C#. Aprender C# também fornece uma base sólida para aprender outras linguagens de programação orientada a objetos.

Caracteristicas do C#:

"C# é uma linguagem orientada a objetos, o que significa que ela segue o paradigma de programação orientada a objetos, o que facilita a organização e manutenção do código."

Variáveis e Tipos de dados:

Para usar uma variável em um programa C#, é necessário declará-la primeiro. A declaração de uma variável, é o processo de informar ao compilador o nome da variável e o tipo de dado que ela armazenará. A sintaxe para declarar uma variável é:


Tipo nomeDaVariavel;

Exemplo:
int idade;
string nome;
Primeiro colocamos qual será o tipo de dados que queremos na variável e logo em seguida colocamos qual será o nome da nossa variável usando a convenção global camelCase.

Tipos de Dados
  • Inteiros: são números inteiros, sem casas decimais. Exemplos: int, long, short.
  • Ponto Flutuante: São números com casas decimais. Exemplos: float, double.
  • Caractere: é um único caractere. Exemplo: char.
  • Booleanos: são valores lógicos, verdadeiro ou falso. Exemplo: bool.
  • Cadeia de caracteres: São sequências de caracteres. Exemplo: String.
Atribuindo Valores às Variáveis

Depois de declarar uma variável, é possível atribuir um valor a ela. A sintaxe para retribuir um valor a ela. A sintaxe para atribuir um valor à uma variável é: nomeDaVariavel = valor; Por exemplo: idade = 25; nome = "João";
